SummaTRON-API

API de acceso al sistema de identificación creada por SummaTRON, tipo 2FA y basado en el Blockchain de TRON para mejorar la identificación de usuarios en las páginas webs.

Existen más de 1.060 millones de páginas web y más de 28 millones de dominos web regitrados. El API de SummaTRON quiere ayudar a simplificar y aumentar la seguridad de las páginas con identificación de usuario.

El API de identificación actua con un sistema 2FA, idenficación de doble factor, basado en que un usuario que posee un cuenta TRON en una wallet y envía un Token ITX a otra cuenta, sólo puede ser él. Si ha perdido el control de su cuenta, ha perdido la identidad y su contenido (equivalente a perder un certificado digital junto a la clave).

En está página puedes probar el funcionamiento. Enviando 1 IdTronix podrás ver la información de su cuenta sin tener que utilizar la clave privada. De esta forma si utiliza Windows no sería necesario teclear nada. Funciona como navegar por Tronscan, pero sin teclear la clave privada. No sustituye la funcionalidad de wallet:envio o pago.

Aquí puedes comprar unos IdTronix para probar el funcionamiento. (1 TRX = 1.000 IdTronix).

Basado en esta premisa, cualquier empresa que posea o quiera implantar un sistema de idenficación de usuarios, puede usar además de usuario y clave: SMS, Google Authenticator, Microsoft Authenticator, etc., SummaTRON será un método similar e incluso más fácil a no tener que esperar a que llegue el SMS, si no que él envía el Token.

Por ejemplo una empresa que quiera usarlo solo tiene que integrar el API y registrase, lo que supone:

  • 1) Crear o disponer de una cuenta TRON, debe comprar 1 SummaTRON y tantos IdTronix que crea necesarios según su número de usuarios, su precio es 0,001 TRX.

  • 2) El proceso de registro consiste en enviar el SummaTRON a la cuenta del proyecto: TFQwLDzUvEc99ktd3TvUc9g3uATGmX2fS7, y muy importe, en la descripción indicar el nombre de la web, por ejemplo:EmpresaXX. Se usará para generar el código para la identificación.

  • 3) Indica a tus usuarios que se instalen una wallet (TronWalletMe tiene la opción PAGAR que agiliza el proceso al scanear todos los datos de una vez).

  • 4) En la instalación de la wallet los usuarios crearán una cuenta TRX.

  • 5) En la web los usuarios deben registar esa cuenta TRX con un dato adicional, será la utilicen para identificarse.

  • 6) La empresa les enviará un número de ITX a dicha cuenta para que el usuario inicialmente no tenga que hace nada, y ya pueda usarlos para identificarse.

  • 7) El usuario cuando quiera entrar en la web (que ya habrá implantado el API de SummaTROn) le aparecerá unos QRs, scanea y envía un ITX que irá a la cuenta de la propia empresa, por tanto la empresa sólo necesitará comprar más ITX si aumentan sus clientes.


Si el Sistema TRON triunfa y el uso de criptomonedas y wallet se generaliza, los mismos usuarios podrán comprarse los ITX, de lo contrario la empresa movería todos los que compre entre sus clientes, necesatará tantos como identificaciones diarias tenga. Mayor actividad, mayor número de ITX que necesita, pero el coste frente a un SMS es de 100.000 menos, ya que los ITX son reutilizables.

Como ejemplos de uso hemos desarrollado un agregador de cuentas TRX, SummaTRON (Summa todo en latín y TRON) y un e-commerce que hemos denominado SHOP donde los usuarios pueden comprar productos digitales mediante TRX y dar de alta sus propios produtos.

Para incorporar el API una página web, debe incluir el siguiente codigo

(También puedes ver toda la información en https://github.com/SummaTron/SummaTRON-API )

En el HEAD 


	<script src="qrcode.min.js" ></script>
	// Qrcode es un desarrollo de https://github.com/davidshimjs/qrcodejs
	<script src="/api/js/apisummatron.js" ></script><br>

En en BODY, en el lugar donde quieras que aparezca los QR para realizar el control de entrada


	<div id="PanelPago" style="display:block;"></div> 
	//Con el css se controla cuando debe estar visible y cuando no.

Al final del HTML se incorpora el código 

	  
	<script>
	   $("#PanelPago").load("/api/PanelPago.html", function() {
	   sCuenta = "XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX";
	   $("#Propietario").text(sCuenta);
	   Verificar(sCuenta,"ES","FUNCION a EJECUTAR");
	   });
	</script>

Con este script se muestra el QR y se indica cual es la cuenta TRON de la empresa.
En FUNCION a EJECUTAR de debe indicar el nombre de la funcion que se ejecutará cuando la identificación finalice, se devuelve como parámetro la cuenta TRON, o un string vacio, si tras 60 segundos no ha se ha producido la identificación.

MUY IMPORTANTE
Para poder utilizar SummaTRON-API debe registrar el nombre de la empresa que será usado como parte de la clave que se envía en el proceso de identificacion.

Este proceso se realiza enviando 1 SummaTRON desde la cuenta de la empresa, la misma que luego se debe incluir en la llamada a la función Verificar(), a la cuenta del proyecto:
TFQwLDzUvEc99ktd3TvUc9g3uATGmX2fS7, y es muy importante, indicar en la Descripcion el nombre de la empresa, sugerimos que sea el dominio sin www ni .com.

¿Cómo veo mis cuentas?

Desde la cuenta de la wallet, debes enviar un token IdTronix (ITX).
Si quieres comprarlos IdTronix puedes hacerlo desde la wallet o desde www.tronscan.org
Para ello puede utilizar los QR que se muestran en la pantalla principal.
TronWallet permite PAGAR scaneando el QR directamente lee todos los datos, en otro caso usa ENVIAR y teclea la Descripción indicada.
Una vez recibido el token se muestra la cartera de cuentas enlazadas.
Sólo el propietario de la Clave Privada puede enviar un token desde una cuenta, así identifica SummaTron a sus usuarios.
Sin tocar el PC o la tablet, el usuario puede acceder a sus cuentas de TRON.
¡Simplemente increíble!, sin identificación ni registro previo. Todo gracias a TRON.NETWORK

¿Cómo enlazar cuentas?

1) Elija una de tus cuentas, que será la principal.
2) Compra tantos SummaTRON (STX) como cuentas tengas.
3) Desde la cuenta principal envía 1 STX a cada cuenta que quieras enlazar.

De esta forma todas tus cuentas quedan enlazadas en el Blockchain de TRON.
En cualquier momento puedes enlazar nuevas cuentas.
Si quieres desenlazar una cuenta, simplemente devuelve el STX a la cuenta principal.
Solamente tus cuentas con al menos 1 STX aparecerán en el informe.

Si necesitas comprar SummaTronix para realizar el proceso de enlazar sus cuentas, puedes comprarlos desde la wallet o desde www.tronscan.org
